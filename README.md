# PhysKit Features
- Grabbing and manipulating objects in a physics based way
- Independently grab objects with your left and right hands
- A physics based character controller

# To Use
- To add the required input mappings, copy the contents of InputMappings.txt, and paste them in your project.godot file under the [Input] heading.
- Add Player.tscn to any existing scene, and add the boolean meta tag "grabbable" (make sure it is set to true) to any rigidbody you want to be able to pick up
